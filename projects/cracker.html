Password Cracker

Source code can be found on the github repository.

Description

I made this project simply because I wanted to create something that was fun and sort of easy. It started out by watching this video by NullByte on YouTube and lead to me wanting to make something similar. I added another layer of cracking not covered by them in their video and added some more code to it to make it user friendly.
It uses a JSON config for things, allowing for you to disable the bruteforce method and just check the dictionary as well as setting character set/password length. I thought it was pretty cool and I put it on Github. I had some unrealistic plans for it (A GUI among other things) and I axed them a while ago and will just add to it wherever I feel lead to.
